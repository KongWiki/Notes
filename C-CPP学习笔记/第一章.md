## 第一章



### int

* int 对float做强行转换时，只做向下取整，例如：

  ```c
  float x = 18.4
  float y = 4.9
  printf("%d", (int)x)
  printf("%d", (int)y)
  ### x = 18
  ### y = 4
  ```

  

### 求余取整

* % 为求余

  ```c
  289%10 = 9
  289%100 = 89			
  ```

  

* / 为取整

  ```c
  289/10 = 28
  289/100 = 2
      
     
  ```

### %的使用

* 每两个为一组

  ```c
  int x = 20;
  printf("%d%%", x);  20%
  printf("%d%%%%", x); 20%%
  printf("%d%%%%%%", x); 20%%%
  ```

* 若为奇数个则向上取整，1个%为0, 3个为一个%以此类推
